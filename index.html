Medical Images

Interpreting the decision making process in classifing Chest XRays

The aim of this project was to implement a computer-aided diagnosis system to make the process of reading and analizing an XRay more efficient for the medical practitioner, and faster for the patients. I built an end-to-end state of the art deep learning pipelines for medical image classification, on a dataset of over 20,000 images. My main contributions consist of data cleansing and preprocessing and VAE-based image segmentation. Classification models based on vision transformers and efficient net. In addition to that, I could address the interpretability Challenges in Chest X-ray Classification by comparing the performances of the models on full images and lung segmented ones. In fact the best results were achieved on non segmented images... Making the model focus on the segmented lungs

Traditional Machine Learning

Winner Determination Problem

Solving the WDP using Tabu Search and Q learning with the use of an early stop strategy to reduce the execution time. Creation of a dataset with random instance generation depending on the number of objects, the number of auctions and the min and max number of objects per auction. The conclusion of the project was that Tabu Search was the more suited algorithm for the problem.
