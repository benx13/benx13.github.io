Papers

High quality ECG dataset based on MIT-BIH recordings for improved heartbeats classification

A.S. Benmessaoud, F. Medjani, Y. Bousseloub, K. Bouaita, D. Benrahem, T. Kezai

The paper proposes a new methodology to create a high-quality ECG heartbeat dataset from the MIT-BIH recordings. The authors use an outlier removal technique and calculate an optimal heartbeat size to avoid mixing successive heartbeats. They also preserve the R-R interval set which is important for diagnosis. The new dataset is publicly available. To evaluate the dataset quality, the authors train a 1D ResNet model which achieves 99.24% accuracy, outperforming prior methods by 5-7%. Downsampling the data further improves model performance and efficiency. Overall, the improved dataset construction technique and deep learning model achieve state-of-the-art heartbeat classification, demonstrating the importance of high-quality training data.

The aim of this project was to implement a computer-aided diagnosis system to make the process of reading and analizing an XRay more efficient for the medical practitioner, and faster for the patients. I built an end-to-end state of the art deep learning pipelines for medical image classification, on a dataset of over 20,000 images. In addition to that, I could address the interpretability Challenges in Chest X-ray Classification by comparing the performances of the models on full images and lung segmented ones. Data Mining

In this project, I implemented algorithms for frequent pattern mining, classification, and clustering on the IBM employee attrition dataset. For frequent pattern mining, I used the Apriori algorithm to find associations. For classification, I compared decision trees, random forests, and XGBoost, tuning hyperparameters and preprocessing data. This project demonstrates my skills in implementing key data mining and machine learning algorithms, properly evaluating models, and building interfaces for model interaction.
